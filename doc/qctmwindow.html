void QCtmWindow::addMoveBar(QWidget *moveBar)

添加移动窗口 moveBar, 移动窗口可通过鼠标拖动来拖动窗口.

Note: 移动窗口必须本身就是本窗口的子窗口.

See also removeMoveBar.

QWidget *QCtmWindow::centralWidget() const

返回中央窗口, 中央窗口默认存在，当中央窗口不存在时返回nullptr.

See also setCentralWidget.

QMenuBar *QCtmWindow::menuBar() const

返回菜单栏, 如果菜单栏不存在，则自动创建并返回一个菜单栏对象.

See also setMenuBar.

QCtmNavigationBar *QCtmWindow::navigationBar() const

返回导航栏, 如果导航栏不存在，将自动创建一个新的导航栏对象并返回.

See also setNavigationBar.

void QCtmWindow::removeMoveBar(QWidget *moveBar)

移除移动窗口 moveBar.

See also addMoveBar.

void QCtmWindow::setCentralWidget(QWidget *widget)

设置中央窗口 widget, 当 widget 为nullptr时删除中央窗口.

See also centralWidget.

void QCtmWindow::setMenuBar(QMenuBar *menuBar)

设置菜单栏 menuBar, 设为nullptr时，删除菜单栏.

See also menuBar.

void QCtmWindow::setNavigationBar(QCtmNavigationBar *bar)

设置导航栏 bar, bar 为nullptr时删除导航栏.

See also navigationBar.

void QCtmWindow::setStatusBar(QStatusBar *statusBar)

设置状态栏 statusBar, 当设置 statusBar 为 nullptr 时将删除状态栏.

See also statusBar.

void QCtmWindow::setTitleBar(QCtmTitleBar *titleBar)

设置标题栏 titleBar, 当 titleBar 为 nullptr 时删除标题栏，如果删除标题栏后导航栏存在，则导航栏替代标题栏响应鼠标拖拽功能.

See also titleBar.

QStatusBar *QCtmWindow::statusBar() const

返回状态栏，如果状态栏对象不存在，QCtmWindow会自动创建一个状态栏对象返回.

See also setStatusBar.

QCtmTitleBar *QCtmWindow::titleBar() const

返回标题栏，当标题栏不存在时自动创建并返回标题栏.

See also setTitleBar.
