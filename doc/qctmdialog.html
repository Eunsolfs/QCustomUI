QCtmDialog Class

这是一个模态窗口，类似于QDialog，去掉了系统的边框和标题栏，用自定义的标题栏替代. More...

QCtmDialog::QCtmDialog(QWidget *parent = nullptr)

构造一个父窗口为 parent 的窗口对象.

void QCtmDialog::addMoveBar(QWidget *moveBar)

添加鼠标可拖拽区域 moveBar, moveBar必须为该窗口的子控件之一.

See also removeMoveBar.

QWidget *QCtmDialog::centralWidget() const

返回中央窗口地址, 如果中央窗口不存在则返回nullptr.

See also setCentralWidget.

void QCtmDialog::removeMoveBar(QWidget *moveBar)

移除鼠标拖拽区域 moveBar.

See also addMoveBar.

void QCtmDialog::setCentralWidget(QWidget *widget)

设置 widget 为窗口的中央窗口, 当 widget 为nullptr时删除中央窗口.

See also centralWidget.

void QCtmDialog::setTitleBar(QCtmTitleBar *titleBar)

设置标题栏 titleBar, 当 titleBar 为 nullptr 时删除标题栏.

See also titleBar.

QCtmTitleBar *QCtmDialog::titleBar() const

返回窗口的标题栏，如果标题栏不存在，自动创建并返回一个新的标题栏.

See also setTitleBar.
